Module 10 Exercise 2

In this exercise, you'll build a web-page that displays multiple pieces of text, and provides an interface (a form) for manipulating the color and fontSize of that text. In order to build and use a reusable chart function, you'll need to work in both the js/ParagraphChart.js and js/main.js files. Let's begin by writing our reusable chart function. This funciton will be called by a selection that has an array of objects bound as it's data. Your function will use the data-join to add a different paragraph (p) for each object in the array. It will also expose methods for setting the color and fontSize of the p elements. In your js/ParagraphChart.js file, do the following:

  • Create a function ParagraphChart that will be your reusable function.
  • Create variables within the function scope to track your color and fontSize.
  • Write your chart function that will be returned by the ParagraphChart function. It should take in a parameter that represents your selection.
  • Inside your chart function, you should use the .each method to loop through your selections (though there will only be one in this case). The .each method takes in a function as it's parameter, and that function's parameter represents the data bound to your selection.
  • Inside the .each function, use D3 to select the current context (i.e., d3.select(this)) as the element in which you want to render your chart. Make sure to store that selecdtion in a variable in which you'll perform your data-join.
  • Using the data parameter of the .each function, perform a data-join on the div element you've selected. As always, you'll need to enter, exit, and update elements. This completes your chart function.
